# Snippet-saver app.

# Web-app that helps to make text/code snippets 
### This app provides interface for making snippets
### Stack:
- Go (http)
- MySQL
- Templates performed

### Deployed and configured, app makes CRUD operations. Making snippets in DB.
### Snippets can be:
- Readed
- Edited
- Deleted
